Алгоритм работы программы:

1. Программа запускается и создает объект для чтения ввода пользователя
2. Входит в бесконечный цикл, пока есть ввод от пользователя
3. Считывает строку (ключ) из консоли
4. Передает ключ в фабрику для создания соответствующего объекта фильма
5. Если фабрика вернула null (неизвестный ключ), программа завершается
6. Если объект создан успешно, выводится название его класса
7. Цикл повторяется для следующего ввода

Логика фабрики:

· "soapOpera" → создает объект SoapOpera
· "cartoon" → создает объект Cartoon
· "thriller" → создает объект Thriller
· Любое другое значение → возвращает null
вывести на экран movie.getClass().getSimpleName().
